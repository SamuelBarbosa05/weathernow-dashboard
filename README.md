# WeatherNow Dashboard

Aplicação web de clima em tempo real desenvolvida com React, Vite e Tailwind CSS.

## Visão geral

O WeatherNow Dashboard permite buscar cidades e visualizar informações climáticas atualizadas, com interface moderna, modo escuro, histórico de buscas e experiência responsiva.

## Funcionalidades

- Busca de clima por cidade
- Exibição de temperatura atual
- Exibição da condição do clima
- Informações de umidade
- Informações de vento
- Histórico de buscas recentes
- Loading animado
- Tratamento de erro para cidades inválidas
- Modo escuro
- Layout responsivo

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- Tailwind CSS
- Axios
- WeatherAPI
