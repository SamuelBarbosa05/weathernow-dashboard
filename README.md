# 🌦️ WeatherNow Dashboard

Modern weather dashboard built with focus on user experience, responsive design and real-time API integration.

## 🔗 Live Demo
[> Add your Vercel link here](https://weathernow-dashboard.vercel.app/)

## 📌 Overview

WeatherNow Dashboard is a modern web application that allows users to search for any city and instantly access real-time weather information through a clean, interactive and responsive interface.

Built to demonstrate practical front-end development skills including API consumption, state management, asynchronous data handling and polished UI implementation.

---

## ✨ Features

✅ Search weather by city  
✅ Real-time temperature data  
✅ Weather condition visualization  
✅ Humidity information  
✅ Wind speed information  
✅ Recent searches history  
✅ Animated loading state  
✅ Error handling for invalid cities  
✅ Dark mode  
✅ Fully responsive layout  
✅ Modern dashboard UI
